## Coffee-Quest

###### COFFE QUEST GAME DESIGN DOCUMENT:


#### Mechanics:
Player gets into the cluster holding a cup of coffee
The cluster has staff walking around with a visual FOV that the player has to avoid
Two UI bars, first bar for coffee temperature, second bar for remaining coffee

#### Game win condition:
- Drinking the entire coffee without getting caught

#### Game over condition:
- _Condition_ 1: Staff sees player -> player drinking coffee -> Chase player -> Game over if caught
- _Condition_ 2: Coffee getting cold before fully drinking it

#### Gameplay:
- Player tries to stay in a safe spot away from stuff, presses a key to start drinking coffee before it gets cold
